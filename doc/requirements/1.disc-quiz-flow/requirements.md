# disc-quiz-flow — 需求规格

## 概述

实现 DISC 测评核心流程：首页（引导）→ 答题页（24 题）→ 结果页（雷达图 + 维度分析）。

## 项目信息

- 项目名: PersonalityTest
- 架构类型: Turborepo Monorepo（npm workspaces）

## 需求版本

| 日期       | 版本 | 说明     |
| ---------- | ---- | -------- |
| 2026-06-14 | v1   | 初始需求 |

## 用户故事

- 作为职场用户，我想要完成 24 道情境题测评，以便了解自己的 DISC 性格类型。
- 作为用户，我想要在首页看到清晰的产品介绍和快速入口，以便快速开始测评。
- 作为用户，我想要在结果页看到雷达图和维度百分比，以便直观理解自己的性格分布。

## 功能需求

### 首页 (Home)
1. [F-001] Hero 区：展示主标题文案"解码行为特质，放大职场影响力"、"开始测评"按钮与"查看历史记录"按钮
2. [F-002] DISC 维度 Bento 磁贴区：以网格卡片介绍 D/I/S/C 四种类型，附带专属渐变图标
3. [F-003] 价值展示区（Benefits）：沟通效果、职业生涯、团队动力三方面价值说明
4. [F-004] 数据指标区（Stats）：展示"1.5万+ 已分析画像"、"98% 准确度评分"、"10分钟 平均测试时长"
5. [F-005] 全局悬浮按钮（FAB）：桌面/移动端右下角常驻"立即开始测评"按钮，带 Tooltip

### 答题页 (Quiz)
6. [F-006] 顶部导航栏：返回上一题按钮 + 当前进度（"第 3 / 24 题"）
7. [F-007] 线性进度条：平滑动效，随答题进度伸缩
8. [F-008] 题目展示：当前题目类别（如 `DECISION MAKING`）+ 情境问题内容
9. [F-009] 单选卡片：4 个选项，点击后高亮，自动标记已选状态
10. [F-010] 底部动作条：帮助/提示按钮 + "下一题"/"提交结果"按钮（未选时禁用）
11. [F-011] 答题状态保护：答题进行中，底部导航栏隐藏（防止误触跳出）

### 结果页 (Result)
12. [F-012] 主导类型展示：性格字母 + 中文名称（如"D (支配型)"）+ 一句话核心画像
13. [F-013] SVG 雷达图：四轴（D/I/S/C）极坐标图，半透明蓝色遮罩，路径描边动画
14. [F-014] 维度百分比条：四色渐变进度条（D=红、I=黄、S=绿、C=蓝），从 0% 增长到目标值动画
15. [F-015] 性格画像深度拆解：核心描述 + 双栏分析板（核心优势 3 条 / 提升领域 3 条）
16. [F-016] CTA 行动区：查看深度解析 / 重新测试 / 查看测试历史 三个按钮

## 非功能需求

- **性能**: 雷达图必须使用原生 SVG，禁止引入第三方图表库
- **动画**: 进度条、选项高亮、雷达图路径描边均需流畅动效（CSS transition / SVG animation）
- **响应式**: Mobile / Tablet / PC 全适配，答题时移动端底部 Tab Bar 隐藏
- **字体**: 英文使用 `Manrope`，标签/代码使用 `JetBrains Mono`
- **图标**: Google `Material Symbols Outlined` 矢量字体

## 功能范围

**In Scope:**
- 首页、答题、结果三个页面的完整 UI 与交互
- DISC 计分算法（各维度次数 / 24 × 100，取整）
- 结果写入 localStorage

**Out of Scope:**
- 深度解析页（feature 2）
- 历史记录管理（feature 2）
- 用户登录/注册（feature 3）
- 云端数据同步（feature 3）
- 支付与报告下载（feature 4）

## 依赖

- `packages/ui` — 共享 shadcn/ui 组件（Button、Card 等）
- TailwindCSS v4（已配置）
- Google Fonts: Manrope + JetBrains Mono + Material Symbols Outlined（需引入）
